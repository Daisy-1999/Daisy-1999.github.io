css特效1（全屏视频背景滚动淡出）

发表于2021-04-17|更新于2021-04-17|css
|阅读量:

背景

我大概描述一下这个结果具体是个什么样子，刚打开这个地址出现的是一个占满窗口的视频，然后在向下滑动的过程中这个视频的颜色会逐渐的变暗，直到消失。下面是具体实现的代码。在这个小案例中我认识了两个css属性。

pointer-events: none;

这个属性的意思是 元素永远不会成为鼠标事件的target

object-fit

object-fit属性指定元素的内容应该如何去适应指定容器的高度与宽度，它一般用于img和video标签，一般可以对这些元素进行保留原始比例的裁剪，缩放或者直接进行拉伸等 。

  • fill:默认，不保证保持原有的比例，内容拉伸填充整个内容容器
  • contain：保持原有比例，内容被缩放
  • cover: 保持原有尺寸比例，但部分内容会被剪切掉
  • none: 保留原有元素内容的长度和宽度，也就是说内容不会被重置
  • scale-dowm: 保持原有尺寸比例
